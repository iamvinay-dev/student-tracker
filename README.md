# Student Tracker 📚

A personal academic companion for students — track attendance, expenses, tasks and generate PDF reports.

## Features
- ✅ Attendance Tracker with subject-wise % and calendar view
- 💰 Money Tracker with income/expense categories
- ☑️ To-Do List with priorities and due dates
- 📅 Daily Planner
- 📊 Expenses PDF Report
- 📈 Attendance PDF Report

## Install as App

1. Open the link in **Chrome on Android**
2. Tap **⋮ menu → Add to Home Screen**
3. Tap **Add** — app icon appears on your home screen!

## Deploy to GitHub Pages

1. Fork or upload files to your GitHub repo
2. Go to **Settings → Pages**
3. Set source to **main branch**
4. Your app will be live at `https://USERNAME.github.io/REPO-NAME`

## Tech Stack
- Pure HTML/CSS/JavaScript — no frameworks
- localStorage for data persistence
- Works completely offline
- jsPDF for PDF generation
